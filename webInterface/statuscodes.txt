ARDUINO MESSAGE STRINGS FORMAT
- ARDUINO MODES STATUS CODES
	- ARDUINO MODE RAW (AR_MR)
	- ARDUINO MODE CALIB (AR_MC)
	- ARDUINO MODE USER (AR_MU)
- ARDUINO CALIBRATION MODE STATUS CODES
	- ARDUINO CALIB SUCCESS (AR_C1)
	- ARDUINO CALIB FAIL (AR_C0)
- ARDUINO CALIBRATION GESTURES DETECTED
	- ARDUINO LEFT (AR_GL)
	- ARDUINO RIGHT (AR_GR)
	- ARDUINO UP (AR_GU)
	- ARDUINO DOWN (AR_GD)
- ARDUINO COMMANDS TO SEND
	- ARDUINO SET MODE USER (AR_SMR)
	- ARDUINO SET MODE RAW (AR_SMR)
	- ARDUINO SET MODE CALIB (AR_SMC)
